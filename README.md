## Contribute

```shell
# Fork.
$ git clone git@github.com:yourusername/chelevra.tmtheme.git
$ cd chelevra.tmtheme/src
# Edit.
$ vi chelevra.json && ./build.sh
# Push
$ git add . && git commit && git push
# Pull Request.
```
